Como surgiu a Internet?

Surgiu na década de 60 como uma rede chamada Arpanet com o intuito de envio de informações entre centro de pesquisas e instalações militares e o Pentágono.

Primeiros Computadores

A evolução dos computadores são divididas no decorrer dos anos em gerações, sendo da primeira até a quinta(atual).

  1. Primeira geração

    2. ENIAC E COLOSSUS: Computadores gigantes que ultilizavam válvulas.

  2. Segunda geração:

    3. É criado o Transistor que conseguia fazer 10 mil vezes mais processamentos que a válvula. Também foi nesse período que o computador foi se tornando menor.

  3. Terceira geração:

    4. Foi criado o primeiro chip integrado diminuindo mais ainda o tamanho dos computadores na época.

  4. Quarta geração:

    5. O microprocessador é criado, com isso vários microcomputadores também.

  5. Quinta geração:

    6. Com a criação do HTML, se inicia a quinta geração de computadores. É lançado o microprocessador Pentium, que avança no processo de músicas e gráficos, com o passar dos anos se tornam máquinas de jogos.

O que é HTML?

Desenvolvido por Tim Berners-Lee, o HTML (HyperText Markup Language, que em português significa Linguagem de Marcação de Hipertexto) é utilizado para estruturar e formatar o conteúdo de páginas web.

Podemos entender a anatomia do HTML como uma árvore, onde a raiz ou pai é o elemento < html> e apartir dela podemos declarar novos elementos que são como ramos ou filhos.

Com o HTML você pode criar vários elementos para construir uma página web com o tema da sua preferência: lojas virtuais, blogs, páginas de notícia etc. Alguns dos principais elementos que podemos criar são: títulos e párafragos, formulários, listas e tabelas, imagens e links, etc.

O que são Tags?

As tags são formadas por uma estrutura própria, começam com o sinal “menor que (<)”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que(>)”. As tags precisam de fechamento, que é definida com a barra de divisão "/". Algumas tags são de auto fechamento por isso não é necessário a barra.

Exemplo:

  1. Tag de fechamento: < h1> Olá mundo< /h1>
  2. Tag de autofechamento: < img src:"banner.jpg">.
